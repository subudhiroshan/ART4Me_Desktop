COM ports are physical ports on the back of a PC that allow serial communication with external devices, like a USB modem in this case.
The system is unable to detect which serial port needs to be accessed and so we have to configure this option manually.



To obtain the COM port information,
  1. Open the 'Device Manager' from the Windows 'Start' menu, by typing Device Manager in the search box.

  2. The Device Manager opens up, and it would appear like this window,



  3. Browse to the 'Ports (COM and LPT)' category and expand it,



  4. Determine the COM port to which our USB modem (Huawei Mobile Connect - 3G PC UI Interface) is connected and select this port in the drop-down menu.
